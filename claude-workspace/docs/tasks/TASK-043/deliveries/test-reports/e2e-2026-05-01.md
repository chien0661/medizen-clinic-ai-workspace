#### Giải thích technical

TASK-033 thay đổi auth model: JWT bây giờ cần `active_clinic_id` claim (lấy từ `account_clinic_role` pivot). Các integration test fixtures tạo users với `user.clinic_id` (legacy field) nhưng không insert vào `account_clinic_role`. Khi login, service lookup `account_clinic_role` → rỗng → `active_clinic_id = None` → tất cả API calls fail với `{"detail":"clinic_id required"}`.

**Action cần thiết:** Update tất cả integration test fixtures để populate `account_clinic_role` sau khi tạo user. Đây là **breaking change có thể dự đoán trước** của TASK-033.
