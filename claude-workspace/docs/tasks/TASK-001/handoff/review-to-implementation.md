## Required Changes (blocking — must fix before re-submit)

### CRITICAL

1. **`tests/conftest.py:17`** — Hardcoded `postgres` hostname will fail in CI (which uses `localhost:5432`).
   **Fix:** `TEST_DATABASE_URL = os.environ.get("DATABASE_URL", "postgresql+asyncpg://cms:cms@localhost:5432/cms")`.

2. **`.dockerignore:22` vs `Dockerfile:28`** — `.dockerignore` excludes `tests/`, but Dockerfile does `COPY tests /app/tests`. Build will fail.
   **Fix:** Remove the `tests/` line from `.dockerignore` (recommended), or drop `COPY tests` and `COPY pyproject.toml` from Dockerfile.

### MAJOR

3. **`app/core/exceptions.py:117-123`** — `unhandled_exception_handler` logs nothing. Every 500 in production will be a black box.
   **Fix:** Add `log.exception("unhandled_exception", path=..., method=..., exc_type=...)` using `app.core.logging.get_logger(__name__)`.

4. **`app/core/base_model.py:12-15`** — `Base` has no `MetaData(naming_convention=...)`. Every autogenerated FK in TASK-002..TASK-024 will violate §2.1 naming.
   **Fix:** Set `metadata = MetaData(naming_convention=NAMING_CONVENTION)` on `Base` (full snippet in review-report.md Issue 4). Then verify migration 0001 still matches (`alembic check` or compare-via-autogen).

5. **`app/core/logging.py:35-42`** — `ProcessorFormatter` chain lacks `format_exc_info` + `dict_tracebacks`. `log.exception(...)` from foreign loggers won't serialize tracebacks to JSON.
   **Fix:** Insert both processors before `renderer` in the formatter's processors list (and also into `shared_processors`).

6. **`tests/conftest.py:20-25`** — Session-scoped `event_loop` fixture is deprecated in pytest-asyncio ≥0.23 and slated for removal.
   **Fix:** Delete the `event_loop` fixture; in `pyproject.toml` add `asyncio_default_fixture_loop_scope = "session"` under `[tool.pytest.ini_options]`. Remove redundant `@pytest.mark.asyncio` decorators from `tests/integration/test_health.py` (auto mode).

### MINOR (fix in same pass)

7. **`tests/unit/test_config.py:8-39`** — All 3 tests inject `Settings(...)` directly; none exercise env loading. Replace with: defaults test + `monkeypatch.setenv` env-loading test + `CORS_ORIGINS` JSON parse test.

10. **`tests/unit/test_config.py:3` and `tests/unit/test_base_model.py:5`** — Unused `import pytest`. Ruff CI lint job will fail (F401). Remove.
